DuplicateFileDeleter / FileCompare
----------
Scan les fichiers d'un dossier et de ses sous-dossiers pour fichier identiques.

* Génère premièrement la liste sans effacer
* Peut aussi trouver les fichiers vide (0byte)
* Détecte les image .jpg illisibles.
* Option pour effacer automatiquement les duplicata (pas de undo!)
